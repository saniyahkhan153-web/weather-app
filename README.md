# 🌤️ Weather App

A beginner-friendly weather application built using HTML, CSS, and JavaScript. The app fetches real-time weather data using the OpenWeatherMap API and displays information such as temperature, weather condition, humidity, wind speed, and pressure.

## ✨ Features

* Search weather by city name
* Real-time weather data
* Dynamic weather icons
* Humidity, Wind, and Pressure details
* Glassmorphism UI Design
* Responsive layout
* Beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
* Font Awesome

## 📸 Preview

A modern weather application with a clean glassmorphism interface and attractive background design.
